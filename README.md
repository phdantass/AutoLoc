# 🚗 AutoLoc  

AutoLoc é uma plataforma automotiva moderna inspirada em aplicações como Waze, Uber, Tesla UI e Google Maps, criada para facilitar a busca por mecânicas, oficinas, locadoras e lojas de peças próximas da localização do usuário.

O projeto foi desenvolvido com foco em experiência do usuário, design premium e navegação intuitiva, oferecendo uma interface futurista e totalmente responsiva para desktop e dispositivos móveis.

---

# ✨ Objetivo do Projeto

O objetivo da AutoLoc é centralizar serviços automotivos em uma única plataforma visual e inteligente, permitindo que o usuário encontre rapidamente:

- Mecânicas próximas
- Oficinas especializadas
- Lojas de peças automotivas
- Serviços de alinhamento e balanceamento
- Elétrica automotiva
- Funilaria
- Locadoras de veículos
- Serviços de pneus e freios

Tudo isso através de uma experiência semelhante a aplicativos modernos de navegação.
